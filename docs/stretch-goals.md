# Stretch Goals

### 1. Add validation rules before submitting the form.

Prevents invalid data from reaching the server and improves user experience.

- Add validation checks in submitForm()
- Show inline error messages.
- Prevent user from submitting form until errant input has been fixed.



### 2. Automatically "refresh" the list of agents

Removes the need for manual reloads and keeps UI in sync.


### 3. Paginate Agent listing

Prevents slowing down when datasets gets large.
